Description of the Project

This is an Angular & Django Project that aims at discovering the basics of those technologies. This project takes into input an image and returns a description of it made via the BLIP image captioning model hosted from hugging face.

The model on hugging face allows for a lightweight project perfect for understanding the basics of Angular and Django. The model could have been hosted on the Django backend but its too heavy for the project. This project also stores informations about the image and its description in a PostgreSQL database.

The project is divided inside Docker containers that are able to achieve different tasks.

The project is divided in two main parts: the frontend and the backend. The frontend is made with Angular and the backend with Django.
